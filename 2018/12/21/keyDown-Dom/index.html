keyDown dom实例
发布于 2018-12-21 / 更新于 2018-12-28 / JavaScript / 热度 0 / 喜欢 0

JQ键盘按下监听事件（1）：

$('body').keydown(function (res){ console.log(res.originalEvent.key); })

以上代码效果为当键盘按键按下的时候打印出来的是按键的名字

比如我在页面上按下”D”的时候 如果大写开着就是“D”，如果没开就是“d”

JQ键盘按下监听事件（2）

$('body').keydown(function (res){ console.log(res.keyCode); })

以上代码效果为打印键盘按键的值;

比如键盘的上下左右键，分别为 37、38、39、40
